Đoàn Thanh Phương - Task 1

  1. So sánh file HTML có định dạng .html và .htm. Nếu không có các thẻ <html> <head> và <body> thì có ảnh hưởng gì trong file HTML không?

    • Hai định dạng .html và .htm về cơ bản là giống nhau nhưng .html được ưu tiên hơn .htm. Đuôi .htm có là do thời xưa phần mở rộng file bao gồm 3 ký tự.
    • Nếu không có các thẻ <html> <head> và <body> thì không ảnh hưởng gì tới file HTML.

  2. Có những cách nào để hiển thị các thẻ HTML để hiển thị ra bên ngoài mà không bị HTML xử lý. Ví dụ hiển thị các nội dung như </br><div> <hl>Hello HTML</hl></div>

    • - Ta sử dụng HTML Entities để hiểm thị hai dấu < > :
      • Dấu < sẽ tương ứng với &lt; hoặc &#60;
      • Dấu > sẽ tương ứng với &gt; hoặc &#62;

  3. Làm thế nào để tạo vùng chọn để định dạng trong CSS cho một tag, một class, một ID?

    • Định dạng CSS cho tag:
      [tên tag] {
      [Nội dung định dạng]
      }
    • Định dạng CSS cho class
      .[tên class] {
      [Nội dung định dạng]
      }
    • Định dạng CSS cho ID:
      #[ID] {
      [Nội dung định dạng]
      }

    • Gạch chân, kẻ đường ngang, chỉ số trên/dưới.
      • Gạch chân: dùng thẻ <u>
      • Đường kẻ ngang: dùng thẻ <hr>
      • Chỉ số trên: dùng thẻ <sup>
      • Chỉ số dưới: dùng thẻ <sub>
